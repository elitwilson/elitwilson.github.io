# ratatui-personal-site

A personal site that looks like a terminal and plays like a tiny game. Built with
[Ratatui], to be delivered to the browser via [Ratzilla] (WASM). Explore a little —
find the key, reach the door — and the content reveals itself.

[Ratatui]: https://ratatui.rs
[Ratzilla]: https://github.com/orhun/ratzilla
